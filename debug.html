Quick Troubleshooting

If you're seeing a blank screen in your app, try these steps:

  1. Open browser developer tools (F12 or Right-click > Inspect)
  2. Check for errors in the Console tab
  3. Verify your React root element exists in the Elements tab
  4. Try adding ?debug=true to your URL

Common Issues

  • Import errors: Make sure all imports use .jsx extension (not .tsx) 
    import Component from './Component.jsx';
  • Missing root element: Ensure index.html has <div id="root"></div>
  • CSS conflicts: Check if any CSS is hiding your content (z-index, display: none, etc.)
  • Routing issues: Ensure React Router is properly configured
